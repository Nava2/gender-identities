JSON Gender Identities is a collection of open source tools to give the users of your website the ability to choose how their gender will be represented from a variety of identities and pronouns.

Usage

Use the files identities.json and pronouns.json to populate your sign up form, user profile page, etc.

Examples

These example scripts will populate a dropdown with that're given an id attribute of identitiesDropdown or pronounsDropdown.

Contributing

Both JSON objects, the source for this website, as well as the example scripts are hosted on GitHub. Please use your talent to help improve the existing content, add scripts for other platforms, and by making sure the lists for gender identities and pronouns is as comprehensive as possible.
